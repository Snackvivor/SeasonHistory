F12: Chinese Poker
2
1
19:33
This poker variation known as Chinese Poker is a spin on the traditional game of poker as it focuses on forming good hands as opposed to betting and bluffing.
19:33
In this game, you will form three sets of hands of 3, 5, 5 cards each instead of the regular 5 card hand with bets seen in Poker.
19:34
You will be facing off 1-v-1 against members of the other tribe. Points are scored based off how many hands are won.
19:34
19:34
This is how the challenge will work:
19:34
Each tribe member will compete directly against a player from the other tribe.
19:35
Both players competing will be given the same set of 17 cards.
19:35
You will each form 3 sets of poker hands based on the following specifications.
19:35
19:35
Normal poker hand hierarchy applies.
19:35
You will form one set of 3 cards and two sets of 5 cards
19:35
For the 3-card hand, the best hand is a ‘Three of a Kind’ (straight and flush does not count)
19:35
19:35
The three sets of hands must adhere to the following restrictions:
19:35
The set of 3 cards must have a lower ranking than the first set of 5, and the second set of 5 cards must have a lower ranking than the second set of 5. So each set gets progressively better.
19:36
For example, if the set of 3 has ‘One Pair’, the first set of 5 has a ‘Straight’ and the second set of 5 has a ‘Flush’, it would be a legal hand. Swapping the sets of 5 would result in an illegal hand.
19:36
If this rule is broken, we will void starting from where it is broken (so if the set of 3 is stronger than either set of 5, the whole hand would be voided).
19:36
19:36
Once you begin, you have a maximum of 7 minutes to form the hand. If you lock in after 7 minutes your submission will not count and we will use the last valid hand submitted within the 7 minutes. If nothing was valid as submitted, we will treat it as if you have no submission and it counts as an auto loss.
19:37
19:37
Scoring is as follows:
19:37
You get +1 point for each set you win and -1 for each set you lose
19:37
If one player wins all 3 sets, they will get +6 and the opponent will get -6 for losing all 3 sets
19:37
The tribe with the highest total score will win immunity.
19:37
If there is a tie, the player with the best overall set will win immunity for their tribe.
19:37
19:37
In your submission channel, type in the command ?f12-<your number> (such as ?f12-1) to receive your cards. You will have to keep track of time on your own, we will not remind you when your 7 minutes are up so watch the clock! (edited)

19:38

19:38
As a tribe please assign your own numbers for the command. (if needed: @Cotija you must sit out 2 players, please decide who they are before assigning any numbers and starting the challenge.) (edited)
